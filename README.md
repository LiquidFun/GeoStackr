# Reddit-GeoStackr-Bot

A bot which posts stacked (added) scores to /r/geoguessr for a single series as specified by the constants at the top of the `GeoStackr` file.

See [this post](https://www.reddit.com/r/geoguessr/comments/j6wzvj/3_october_streak_stacker_2/g87hqfh?utm_source=share&utm_medium=web2x&context=3) for an example.

Reddit API key required, which is saved in the RedditAPIAccess.txt file in this format:

```
client_id
client_secret
username
password
```
